## Application Info
React-Native 기반의 TUK & Gtec의 통학 버스 스케줄 애플리케이션

다음과 같은 기능을 포함하고 있다.
1. 서버에서 버스 시간표와 지하철 실시간 위치 정보를 받아와 표시
2. 구글 맵을 사용하여 탑승 위치에 대한 GPS 표시
3. 로그인 사용자가 이용할 수 있는 socket.io 기반의 채팅 기능
